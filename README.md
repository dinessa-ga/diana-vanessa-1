
# Diana Vanessa Portfolio

A modern, interactive portfolio website showcasing my work as a Tech Content Strategist. This project highlights branding projects, case studies, skills, and services, built with a focus on clean design and user experience.

This is a code bundle for Portfolio-2. The original project is available at [https://www.figma.com/design/F7XOjajL2NeakHaRuHylMw/Portfolio-2](https://www.figma.com/design/F7XOjajL2NeakHaRuHylMw/Portfolio-2).

## Features

- **Hero Section**: Eye-catching introduction with key highlights.
- **Branding Showcase**: Display of featured projects with images, colors, and services.
- **About Section**: Personal overview and background.
- **Skills Section**: Showcase of technical and creative competencies.
- **Contact Form**: Easy way for visitors to get in touch.
- **Case Studies**: Detailed project breakdowns with client overviews, challenges, strategies, and results.
- **Dark/Light Theme Toggle**: User-friendly theme switching.
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.
- **Hash-Based Routing**: Seamless navigation between home and individual case studies.

## Tech Stack

- **Frontend Framework**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS with custom utilities
- **UI Components**: Radix UI primitives for accessibility
- **Icons**: Lucide React
- **Animations**: Motion library
- **Theme Management**: next-themes
- **Charts**: Recharts (if used in skills or data visualization)
- **Forms**: React Hook Form with validation
